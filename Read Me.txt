Simple Calculator
Description
This is a simple calculator application written in C that performs basic arithmetic operations: addition, subtraction, multiplication, and division.

Requirements
- C compiler (e.g., gcc)

Instructions to Run the Calculator
1. Compile the code:
   gcc calculator.c -o calculator
2. Run the executable:
   ./calculator
